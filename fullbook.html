ארבע פעולות החשבון

ארבע פעולות החשבון הבסיסיות

חשבון הוא הבסיס לכל המתמטיקה. ארבע הפעולות הבסיסיות הן:

  • חיבור (+): הוספת מספרים
  • חיסור (−): הפחתת מספרים
  • כפל (×): הכפלת מספרים
  • חילוק (÷): חלוקת מספרים

סדר פעולות (סדר עדיפויות)

בביטוי מתמטי מורכב, נתבצע את הפעולות בסדר הבא:

  1. סוגריים - מה שבתוך הסוגריים קודם
  2. חזקות ושורשים
  3. כפל וחילוק (משמאל לימין)
  4. חיבור וחיסור (משמאל לימין)
דוגמה מפורטת:

חשב: \(2 + 3 \times 4 - (8 ÷ 2)\)

פתרון שלב אחר שלב:

שלב 1: סוגריים - \(8 ÷ 2 = 4\)

שלב 2: כפל - \(3 \times 4 = 12\)

שלב 3: חיבור וחיסור משמאל לימין:

\(2 + 12 - 4 = 14 - 4 = 10\)

תרגילים לתרגול
תרגיל 1: חשב: \(15 + 8 - 6\)
תרגיל 2: חשב: \(7 \times 6 ÷ 2\)
תרגיל 3: חשב: \(3 + 4 \times 2\)
תרגיל 4: חשב: \((5 + 3) \times 2\)
תרגיל 5: חשב: \(20 - 12 ÷ 4 + 3\)
תרגיל 6: חשב: \(6 \times (4 + 2) - 10\)
תרגיל 7: חשב: \(100 ÷ 5 ÷ 4\)
תרגיל 8: חשב: \(2 + 3 \times (4 - 1) + 5\)

תשובות:

תרגיל 1: \(15 + 8 - 6 = 23 - 6 = 17\)
תרגיל 2: \(7 \times 6 ÷ 2 = 42 ÷ 2 = 21\)
תרגיל 3: לפי סדר פעולות: \(3 + 4 \times 2 = 3 + 8 = 11\)
תרגיל 4: תחילה סוגריים: \((5 + 3) \times 2 = 8 \times 2 = 16\)
תרגיל 5: תחילה חילוק: \(20 - 3 + 3 = 20\)
תרגיל 6: \(6 \times 6 - 10 = 36 - 10 = 26\)
תרגיל 7: משמאל לימין: \(100 ÷ 5 ÷ 4 = 20 ÷ 4 = 5\)
תרגיל 8: \(2 + 3 \times 3 + 5 = 2 + 9 + 5 = 16\)
שברים ומספרים עשרוניים

עבודה עם שברים

שבר מייצג חלק מתוך שלם. השבר \(\frac{a}{b}\) אומר שחילקנו את השלם ל-b חלקים שווים ולקחנו a חלקים.

פעולות בשברים

חיבור וחיסור שברים

כדי לחבר או לחסר שברים, צריך למצוא מכנה משותף:

\(\frac{a}{c} + \frac{b}{c} = \frac{a+b}{c}\)
כפל שברים
\(\frac{a}{b} \times \frac{c}{d} = \frac{a \times c}{b \times d}\)
חילוק שברים
\(\frac{a}{b} ÷ \frac{c}{d} = \frac{a}{b} \times \frac{d}{c}\)

מספרים עשרוניים

מספר עשרוני הוא דרך נוספת לכתוב שבר. למשל: \(0.5 = \frac{1}{2}\)

דוגמה מפורטת:

חשב: \(\frac{2}{3} + \frac{1}{4}\)

פתרון:

המכנה המשותף הקטן של 3 ו-4 הוא 12:

\(\frac{2}{3} = \frac{2 \times 4}{3 \times 4} = \frac{8}{12}\)

\(\frac{1}{4} = \frac{1 \times 3}{4 \times 3} = \frac{3}{12}\)

\(\frac{8}{12} + \frac{3}{12} = \frac{11}{12}\)

תרגילים לתרגול
תרגיל 1: חשב: \(\frac{1}{2} + \frac{1}{3}\)
תרגיל 2: חשב: \(\frac{3}{4} \times \frac{2}{5}\)
תרגיל 3: חשב: \(\frac{2}{3} - \frac{1}{6}\)
תרגיל 4: חשב: \(\frac{1}{2} ÷ \frac{1}{4}\)
תרגיל 5: המר לשבר: \(0.75\)
תרגיל 6: חשב: \(\frac{5}{6} + \frac{1}{3}\)
תרגיל 7: חשב: \(2\frac{1}{4} + 1\frac{1}{2}\)
תרגיל 8: המר למספר עשרוני: \(\frac{3}{8}\)

תשובות:

תרגיל 1: מכנה משותף 6: \(\frac{3}{6} + \frac{2}{6} = \frac{5}{6}\)
תרגיל 2: \(\frac{3 \times 2}{4 \times 5} = \frac{6}{20} = \frac{3}{10}\)
תרגיל 3: מכנה משותף 6: \(\frac{4}{6} - \frac{1}{6} = \frac{3}{6} = \frac{1}{2}\)
תרגיל 4: \(\frac{1}{2} ÷ \frac{1}{4} = \frac{1}{2} \times \frac{4}{1} = 2\)
תרגיל 5: \(0.75 = \frac{75}{100} = \frac{3}{4}\)
תרגיל 6: מכנה משותף 6: \(\frac{5}{6} + \frac{2}{6} = \frac{7}{6}\)
תרגיל 7: \(2\frac{1}{4} + 1\frac{2}{4} = 3\frac{3}{4}\)
תרגיל 8: \(\frac{3}{8} = 3 ÷ 8 = 0.375\)
אחוזים ויחסים

אחוזים

אחוז הוא דרך לבטא חלק מתוך 100. הסימן % אומר "מתוך מאה".

\(x\% = \frac{x}{100}\)

\(\text{אחוז} = \frac{\text{חלק}}{\text{שלם}} \times 100\)

יחסים ופרופורציות

יחס מבטא את הקשר בין שני גדלים. פרופורציה הוא שוויון בין שני יחסים.

\(\frac{a}{b} = \frac{c}{d}\) (פרופורציה)
דוגמה:

מה זה 25% מ-80?

פתרון:

\(25\% \text{ של } 80 = \frac{25}{100} \times 80 = 0.25 \times 80 = 20\)

תרגילים לתרגול
תרגיל 1: מה זה 20% מ-50?
תרגיל 2: 15 הוא כמה אחוזים מ-60?
תרגיל 3: במחלקה 24 תלמידים, מתוכם 18 בנים. כמה אחוזים הן הבנות?
תרגיל 4: מוצר עלה 200 ש"ח ויש עליו הנחה של 15%. מה המחיר אחרי הנחה?
תרגיל 5: פתור פרופורציה: \(\frac{3}{4} = \frac{x}{12}\)

תשובות:

תרגיל 1: \(\frac{20}{100} \times 50 = 0.2 \times 50 = 10\)
תרגיל 2: \(\frac{15}{60} \times 100 = \frac{1}{4} \times 100 = 25\%\)
תרגיל 3: 6 בנות מתוך 24: \(\frac{6}{24} \times 100 = 25\%\)
תרגיל 4: הנחה: \(200 \times 0.15 = 30\) ש"ח. מחיר אחרי הנחה: \(200 - 30 = 170\) ש"ח
תרגיל 5: \(3 \times 12 = 4 \times x\), \(36 = 4x\), \(x = 9\)

משוואות ליניאריות בנעלם אחד

משוואות ליניאריות

משוואה ליניארית בנעלם אחד היא משוואה בצורה:

\(ax + b = c\)

כאשר a, b, c הם מספרים ידועים ו-x הוא הנעלם.

שלבי פתרון

  1. בודד את האיבר עם הנעלם בצד אחד
  2. העבר את כל המספרים לצד השני
  3. חלק במקדם של הנעלם
דוגמה:

פתור: \(2x + 3 = 11\)

פתרון:

שלב 1: העבר את 3 לצד ימין:

\(2x = 11 - 3\)

\(2x = 8\)

שלב 2: חלק ב-2:

\(x = \frac{8}{2} = 4\)

בדיקה: \(2(4) + 3 = 8 + 3 = 11\) ✓

תרגילים לתרגול
תרגיל 1: פתור: \(3x + 5 = 14\)
תרגיל 2: פתור: \(2x - 7 = 5\)
תרגיל 3: פתור: \(4x + 1 = 3x + 8\)
תרגיל 4: פתור: \(\frac{x}{3} + 2 = 5\)
תרגיל 5: פתור: \(5(x - 2) = 15\)
תרגיל 6: פתור: \(7x - 3 = 4x + 12\)
תרגיל 7: פתור: \(\frac{2x - 1}{3} = 7\)
תרגיל 8: פתור: \(0.5x + 1.5 = 4\)

תשובות:

תרגיל 1: \(3x = 14 - 5 = 9\), לכן \(x = \frac{9}{3} = 3\). בדיקה: \(3(3) + 5 = 14\) ✓
תרגיל 2: \(2x = 5 + 7 = 12\), לכן \(x = \frac{12}{2} = 6\). בדיקה: \(2(6) - 7 = 5\) ✓
תרגיל 3: \(4x - 3x = 8 - 1\), לכן \(x = 7\). בדיקה: \(4(7) + 1 = 29\) ו-\(3(7) + 8 = 29\) ✓
תרגיל 4: \(\frac{x}{3} = 5 - 2 = 3\), לכן \(x = 3 \times 3 = 9\). בדיקה: \(\frac{9}{3} + 2 = 5\) ✓
תרגיל 5: \(5x - 10 = 15\), \(5x = 25\), לכן \(x = 5\). בדיקה: \(5(5-2) = 15\) ✓
תרגיל 6: \(7x - 4x = 12 + 3\), \(3x = 15\), לכן \(x = 5\)
תרגיל 7: \(2x - 1 = 21\), \(2x = 22\), לכן \(x = 11\)
תרגיל 8: \(0.5x = 4 - 1.5 = 2.5\), לכן \(x = \frac{2.5}{0.5} = 5\)
אי-שוויונות ליניאריות

אי-שוויונות ליניאריות

אי-שוויון ליניארי דומה למשוואה, אך במקום שוויון יש אי-שוויון:

\(ax + b < c\) או \(ax + b > c\)

כללי פתרון

  • ניתן לחבר או לחסר מספר משני הצדדים
  • ניתן לכפול או לחלק במספר חיובי - הכיוון נשמר
  • חשוב! בכפל או חלוקה במספר שלילי - הכיוון מתהפך
דוגמה:

פתור: \(3x - 2 \leq 7\)

פתרון:

\(3x \leq 7 + 2\)

\(3x \leq 9\)

\(x \leq 3\)

בדיקה: עבור \(x = 2\): \(3(2) - 2 = 4 \leq 7\) ✓

תרגילים לתרגול
תרגיל 1: פתור: \(2x + 1 > 9\)
תרגיל 2: פתור: \(-x + 5 \geq 2\)
תרגיל 3: פתור: \(5x - 3 < 2x + 9\)
תרגיל 4: פתור: \(\frac{x - 1}{2} \geq 3\)
תרגיל 5: פתור: \(-2x + 8 > 12\)
תרגיל 6: פתור: \(3(x + 1) \leq 2x + 7\)

תשובות:

תרגיל 1: \(2x > 9 - 1 = 8\), לכן \(x > 4\)
תרגיל 2: \(-x \geq 2 - 5 = -3\), כפל ב-(-1): \(x \leq 3\) (הכיוון התהפך)
תרגיל 3: \(5x - 2x < 9 + 3\), \(3x < 12\), לכן \(x < 4\)
תרגיל 4: \(x - 1 \geq 6\), לכן \(x \geq 7\)
תרגיל 5: \(-2x > 4\), חלוקה ב-(-2): \(x < -2\) (הכיוון התהפך)
תרגיל 6: \(3x + 3 \leq 2x + 7\), \(x \leq 4\)

משוואות ריבועיות

משוואות ריבועיות

משוואה ריבועית היא משוואה בצורה:

\(ax^2 + bx + c = 0\)

כאשר \(a \neq 0\).

נוסחת השורשים

הפתרון הכללי של משוואה ריבועית הוא:

\(x_{1,2} = \frac{-b \pm \sqrt{b^2-4ac}}{2a}\)

הדיסקרימיננטה

הדיסקרימיננטה \(\Delta = b^2 - 4ac\) קובעת את מספר הפתרונות:

  • אם \(\Delta > 0\): שני פתרונות שונים
  • אם \(\Delta = 0\): פתרון יחיד
  • אם \(\Delta < 0\): אין פתרונות ממשיים

פירוק לגורמים

לעיתים ניתן לפתור על ידי פירוק: \(ax^2 + bx + c = a(x-x_1)(x-x_2)\)

דוגמה:

פתור: \(x^2 - 5x + 6 = 0\)

פתרון:

\(a = 1, b = -5, c = 6\)

\(\Delta = (-5)^2 - 4 \cdot 1 \cdot 6 = 25 - 24 = 1\)

\(x_{1,2} = \frac{5 \pm \sqrt{1}}{2} = \frac{5 \pm 1}{2}\)

\(x_1 = 3, x_2 = 2\)

תרגילים לתרגול
תרגיל 1: פתור: \(x^2 - 7x + 12 = 0\)
תרגיל 2: חשב את הדיסקרימיננטה של \(2x^2 + 3x + 5 = 0\)
תרגיל 3: פתור: \(x^2 + 6x + 9 = 0\)
תרגיל 4: פתור: \(x^2 - 4 = 0\)
תרגיל 5: פתור: \(2x^2 - 8x + 6 = 0\)
תרגיל 6: פתור: \(x^2 + 2x - 8 = 0\)
תרגיל 7: עבור איזה ערך של k למשוואה \(x^2 + 4x + k = 0\) יש פתרון יחיד?
תרגיל 8: פתור: \(3x^2 - 12x = 0\)
תרגיל 9: מצא את סכום השורשים של \(2x^2 - 5x + 1 = 0\)

תשובות:

תרגיל 1: \(\Delta = 49 - 48 = 1\), \(x = \frac{7 \pm 1}{2}\), לכן \(x_1 = 4, x_2 = 3\)
תרגיל 2: \(\Delta = 3^2 - 4 \cdot 2 \cdot 5 = 9 - 40 = -31\)
תרגיל 3: \(\Delta = 36 - 36 = 0\), פתרון יחיד: \(x = \frac{-6}{2} = -3\)
תרגיל 4: \(x^2 = 4\), לכן \(x = \pm 2\)
תרגיל 5: חלוקה ב-2: \(x^2 - 4x + 3 = 0\). פירוק: \((x-1)(x-3) = 0\), לכן \(x = 1\) או \(x = 3\)
תרגיל 6: \(\Delta = 4 + 32 = 36\), \(x = \frac{-2 \pm 6}{2}\), לכן \(x_1 = 2, x_2 = -4\)
תרגיל 7: פתרון יחיד כאשר \(\Delta = 0\): \(16 - 4k = 0\), לכן \(k = 4\)
תרגיל 8: הוצאת גורם משותף: \(3x(x - 4) = 0\), לכן \(x = 0\) או \(x = 4\)
תרגיל 9: לפי נוסחאות ויאטה: סכום השורשים = \(\frac{-b}{a} = \frac{5}{2}\)
פרבולות ופונקציות ריבועיות

פונקציות ריבועיות

פונקציה ריבועית היא פונקציה בצורה:

\(f(x) = ax^2 + bx + c\)

תכונות הפרבולה

  • קודקוד: \(x_v = -\frac{b}{2a}\), \(y_v = f(x_v)\)
  • ציר סימטריה: \(x = -\frac{b}{2a}\)
  • כיוון הפתיחה: מעלה אם \(a > 0\), מטה אם \(a < 0\)
צורה קנונית:
\(f(x) = a(x - h)^2 + k\)

כאשר הקודקוד הוא \((h, k)\)

דוגמה:

מצא את קודקוד הפרבולה \(f(x) = x^2 - 4x + 5\)

פתרון:

\(x_v = -\frac{-4}{2 \cdot 1} = 2\)

\(y_v = f(2) = 4 - 8 + 5 = 1\)

הקודקוד: \((2, 1)\)

תרגילים לתרגול
תרגיל 1: מצא את קודקוד הפרבולה \(f(x) = x^2 + 6x + 8\)
תרגיל 2: עבור \(f(x) = -2x^2 + 8x - 6\), האם הפרבולה פתוחה מעלה או מטה?
תרגיל 3: כתוב בצורה קנונית: \(f(x) = x^2 - 4x + 7\)
תרגיל 4: מצא את ציר הסימטריה של \(f(x) = 3x^2 - 12x + 5\)
תרגיל 5: מצא את הערך המינימלי של \(f(x) = x^2 - 6x + 10\)

תשובות:

תרגיל 1: \(x_v = -\frac{6}{2} = -3\), \(y_v = 9 - 18 + 8 = -1\). קודקוד: \((-3, -1)\)
תרגיל 2: \(a = -2 < 0\), לכן הפרבולה פתוחה מטה
תרגיל 3: השלמה לריבוע מושלם: \(f(x) = (x-2)^2 + 3\)
תרגיל 4: ציר סימטריה: \(x = -\frac{-12}{2 \cdot 3} = 2\)
תרגיל 5: קודקוד בנקודה \(x = 3\), \(f(3) = 9 - 18 + 10 = 1\). מינימום: 1

משולשים ותכונותיהם

משולשים

משולש הוא צורה בעלת שלוש צלעות ושלוש זוויות.

סוגי משולשים לפי צלעות:

  • משולש שווה צלעות: כל הצלעות שוות
  • משולש שווה שוקיים: שתי צלעות שוות
  • משולש שונה צלעות: כל הצלעות שונות

סוגי משולשים לפי זוויות:

  • משולש חד זוויתי: כל הזוויות חדות (< 90°)
  • משולש ישר זוויתי: זווית אחת ישרה (= 90°)
  • משולש קהה זוויתי: זווית אחת קהה (> 90°)
נוסחאות חשובות:

סכום זוויות המשולש: \(\alpha + \beta + \gamma = 180°\)

שטח משולש: \(S = \frac{1}{2} \cdot \text{בסיס} \cdot \text{גובה}\)

משפט פיתגורס: \(a^2 + b^2 = c^2\) (במשולש ישר זוויתי)

דוגמה:

במשולש ABC: ∠A = 60°, ∠B = 50°. מצא את ∠C.

פתרון:

\(\angle C = 180° - 60° - 50° = 70°\)

תרגילים לתרגול
תרגיל 1: במשולש זוויות 45° ו-70°. מצא את הזווית השלישית.
תרגיל 2: משולש ישר זוויתי עם רגליים באורכים 3 ו-4. מצא את היתר.
תרגיל 3: במשולש שווה שוקיים הזווית שבקודקוד היא 40°. מצא את זוויות הבסיס.
תרגיל 4: שטח משולש עם בסיס 8 ס"מ וגובה 6 ס"מ.

מעגל הוא קבוצת כל הנקודות במישור הנמצאות במרחק קבוע מנקודה נתונה.

נוסחאות המעגל:

שטח: \(A = \pi r^2\)

גאומטריה אנליטית

\(d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}\)

\(m = \frac{y_2-y_1}{x_2-x_1}\)

\(\sin \alpha = \frac{\text{תידגנ עלצ}}{\text{רתי}}\)
\(\cos \alpha = \frac{\text{הכומס עלצ}}{\text{רתי}}\)
\(\tan \alpha = \frac{\text{תידגנ עלצ}}{\text{הכומס עלצ}}\)

זוויות מיוחדות

זווית sin cos tan
30° \(\frac{1}{2}\) \(\frac{\sqrt{3}}{2}\) \(\frac{1}{\sqrt{3}}\)
45° \(\frac{\sqrt{2}}{2}\) \(\frac{\sqrt{2}}{2}\) \(1\)

\(\sin^2 \alpha + \cos^2 \alpha = 1\) (זהות פיתגורס)
\(\tan \alpha = \frac{\sin \alpha}{\cos \alpha}\)

הסתברות קלאסית

ההסתברות של אירוע A היא:

\(P(A) = \frac{\text{תובוטה תואצותה רפסמ}}{\text{תוירשפאה תואצותה רפסמ}}\)

תכונות בסיסיות

  • \(0 \leq P(A) \leq 1\)
  • \(P(A^c) = 1 - P(A)\) (אירוע משלים)
  • \(P(A \cup B) = P(A) + P(B) - P(A \cap B)\)
  • אירועים בלתי תלויים: \(P(A \cap B) = P(A) \cdot P(B)\)

הסתברות מותנית

\(P(A|B) = \frac{P(A \cap B)}{P(B)}\)
דוגמה:

מה ההסתברות לקבלת מספר זוגי בהטלת קובייה?

פתרון:

מספרים זוגיים: 2, 4, 6 (3 אפשרויות)

סה"כ אפשרויות: 6

\(P = \frac{3}{6} = \frac{1}{2}\)

תרגיל 3: 5 כדורים אדומים מתוך 8 כדורים: \(P = \frac{5}{8}\)
תרגיל 4: \(P(A^c) = 1 - P(A) = 1 - 0.3 = 0.7\)

תרגיל 6: אפשרויות: EE, EF, FE, FF. לפחות עץ אחד: EE, EF, FE (3 מתוך 4). \(P = \frac{3}{4}\)
תרגיל 7: עבור אירועים בלתי תלויים: \(P(A \cap B) = P(A) \cdot P(B) = 0.4 \times 0.3 = 0.12\)
תרגיל 8: 4 מלכים + 4 אסים = 8 קלפים מתוך 52. \(P = \frac{8}{52} = \frac{2}{13}\)
קומבינטוריקה

ספירה וסידורים

קומבינטוריקה עוסקת בספירת אפשרויות ודרכי סידור.

עקרון הכפל

אם יש m דרכים לבצע פעולה ראשונה ו-n דרכים לבצע פעולה שנייה, אז יש \(m \times n\) דרכים לבצע את שתיהן.

פקטוריאל

\(n! = n \times (n-1) \times (n-2) \times ... \times 2 \times 1\)

תמורות וצירופים

תמורות (הסדר חשוב): \(P(n,r) = \frac{n!}{(n-r)!}\)

צירופים (הסדר לא חשוב): \(C(n,r) = \binom{n}{r} = \frac{n!}{r!(n-r)!}\)

דוגמה:

בכמה דרכים ניתן לסדר 5 ספרים על מדף?

פתרון:

\(5! = 5 \times 4 \times 3 \times 2 \times 1 = 120\) דרכים

תרגילים לתרגול
תרגיל 1: חשב \(4!\)
תרגיל 2: בכמה דרכים ניתן לבחור 3 תלמידים מתוך 10?
תרגיל 3: בכמה דרכים ניתן לסדר 3 אנשים ברצף מתוך 5?
תרגיל 4: כמה מספרים שונים בני 3 ספרות ניתן ליצור מהספרות 1,2,3,4,5 (ללא חזרה)?
תרגיל 5: כמה דרכים יש לבחור נשיא וסגן מתוך 8 מועמדים?

תשובות:

תרגיל 1: \(4! = 4 \times 3 \times 2 \times 1 = 24\)
תרגיל 2: \(\binom{10}{3} = \frac{10!}{3! \times 7!} = \frac{10 \times 9 \times 8}{3 \times 2 \times 1} = 120\)
תרגיל 3: \(P(5,3) = \frac{5!}{2!} = \frac{120}{2} = 60\)
תרגיל 4: \(P(5,3) = 5 \times 4 \times 3 = 60\)
תרגיל 5: 8 בחירות לנשיא ו-7 לסגן: \(8 \times 7 = 56\)

נגזרות של פולינומים

חישוב נגזרות

הנגזרת מודדת את קצב השינוי של פונקציה בנקודה נתונה.

כללים בסיסיים

  • \(\frac{d}{dx}(c) = 0\) (נגזרת קבוע)
  • \(\frac{d}{dx}(x^n) = nx^{n-1}\) (כלל החזקה)
  • \(\frac{d}{dx}(f+g) = f' + g'\) (נגזרת סכום)
  • \(\frac{d}{dx}(cf) = c \cdot f'\) (נגזרת כפולה בקבוע)

פונקציות מיוחדות

\(\frac{d}{dx}(\sin x) = \cos x\)

\(\frac{d}{dx}(\cos x) = -\sin x\)

\(\frac{d}{dx}(e^x) = e^x\)

\(\frac{d}{dx}(\ln x) = \frac{1}{x}\)

דוגמה מפורטת:

מצא את הנגזרת של \(f(x) = 3x^4 - 2x^3 + 5x - 7\)

פתרון שלב אחר שלב:

\(f'(x) = \frac{d}{dx}(3x^4) - \frac{d}{dx}(2x^3) + \frac{d}{dx}(5x) - \frac{d}{dx}(7)\)

\(f'(x) = 3 \cdot 4x^3 - 2 \cdot 3x^2 + 5 \cdot 1 - 0\)

\(f'(x) = 12x^3 - 6x^2 + 5\)

תרגילים לתרגול
תרגיל 1: מצא את הנגזרת של \(g(x) = x^3 - 4x + 1\)
תרגיל 2: מצא את הנגזרת של \(h(x) = 5x^4 + 2x^2 - 8\)
תרגיל 3: מצא את הנגזרת של \(f(x) = \frac{1}{x^2}\)
תרגיל 4: מצא את הנגזרת של \(y = 3\sqrt{x} + \frac{2}{x}\)
תרגיל 5: מצא את הנגזרת של \(f(x) = (2x + 1)^3\) (השתמש בכלל השרשרת)
תרגיל 6: מה הנגזרת של \(f(x) = x^5 - 3x^3 + 7x - 2\) בנקודה \(x = 1\)?
תרגיל 7: מצא את הנגזרת השנייה של \(f(x) = x^4 - 2x^2\)

תשובות:

תרגיל 1: \(g'(x) = 3x^2 - 4 + 0 = 3x^2 - 4\)
תרגיל 2: \(h'(x) = 5 \cdot 4x^3 + 2 \cdot 2x - 0 = 20x^3 + 4x\)
תרגיל 3: \(f(x) = x^{-2}\), לכן \(f'(x) = -2x^{-3} = -\frac{2}{x^3}\)
תרגיל 4: \(y = 3x^{1/2} + 2x^{-1}\), לכן \(y' = 3 \cdot \frac{1}{2}x^{-1/2} - 2x^{-2} = \frac{3}{2\sqrt{x}} - \frac{2}{x^2}\)
תרגיל 5: יהי \(u = 2x + 1\), אז \(f(x) = u^3\). \(\frac{df}{dx} = 3u^2 \cdot \frac{du}{dx} = 3(2x+1)^2 \cdot 2 = 6(2x+1)^2\)
תרגיל 6: \(f'(x) = 5x^4 - 9x^2 + 7\). בנקודה \(x = 1\): \(f'(1) = 5 - 9 + 7 = 3\)
תרגיל 7: \(f'(x) = 4x^3 - 4x\), \(f''(x) = 12x^2 - 4\)
אינטגרלים בסיסיים

אינטגרלים

האינטגרל הוא הפעולה ההפוכה לגזירה, ומחשב את השטח מתחת לגרף.

אינטגרל לא מסוים

\(\int x^n dx = \frac{x^{n+1}}{n+1} + C\) (כאשר \(n \neq -1\))

אינטגרלים בסיסיים

\(\int 1 dx = x + C\)

\(\int x dx = \frac{x^2}{2} + C\)

\(\int \sin x dx = -\cos x + C\)

\(\int \cos x dx = \sin x + C\)

\(\int e^x dx = e^x + C\)

אינטגרל מסוים

\(\int_a^b f(x) dx = F(b) - F(a)\)

כאשר F הוא הפונקציה הקדומה של f.

דוגמה:

חשב את \(\int (3x^2 + 2x - 1) dx\)

פתרון:

\(\int (3x^2 + 2x - 1) dx = 3 \cdot \frac{x^3}{3} + 2 \cdot \frac{x^2}{2} - x + C\)

\(= x^3 + x^2 - x + C\)

תרגילים לתרגול
תרגיל 1: חשב \(\int x^4 dx\)
תרגיל 2: חשב \(\int (2x + 3) dx\)
תרגיל 3: חשב \(\int_0^2 x dx\)
תרגיל 4: חשב \(\int (x^3 - 4x + 5) dx\)
תרגיל 5: חשב \(\int_1^3 (2x - 1) dx\)
תרגיל 6: מה הפונקציה הקדומה של \(f(x) = 6x^2 - 4x + 1\)?

תשובות:

תרגיל 1: \(\int x^4 dx = \frac{x^5}{5} + C\)
תרגיל 2: \(\int (2x + 3) dx = 2 \cdot \frac{x^2}{2} + 3x + C = x^2 + 3x + C\)
תרגיל 3: \(\int_0^2 x dx = \left[\frac{x^2}{2}\right]_0^2 = \frac{4}{2} - 0 = 2\)
תרגיל 4: \(\int (x^3 - 4x + 5) dx = \frac{x^4}{4} - 2x^2 + 5x + C\)
תרגיל 5: \(\int_1^3 (2x - 1) dx = [x^2 - x]_1^3 = (9 - 3) - (1 - 1) = 6\)
תרגיל 6: \(F(x) = \int (6x^2 - 4x + 1) dx = 2x^3 - 2x^2 + x + C\)
