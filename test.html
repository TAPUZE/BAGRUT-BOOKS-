מיסודות החשבון לחשבון דיפרנציאלי

הוראות לשימוש

חוברת תרגילים זו מיועדת לתרגול בצמוד לספר המתמטיקה הראשי. כל פרק כולל תזכורת מהירה של הנושא ותרגילים מדורגים לפי רמת קושי. השתמשו בחללים המיועדים לכתיבת התשובות והפתרונות. מפתח התשובות נמצא בסוף החוברת. בהצלחה!

9. \((4 + 2) \times 3 = 6 \times 3 = 18\)
10. \(45 - 3 \times 10 ÷ 5 = 45 - 30 ÷ 5 = 45 - 6 = 39\)

שברים ומספרים עשרוניים

1. \(\frac{2}{5} + \frac{1}{5} = \frac{3}{5}\)
2. \(\frac{3 \times 2}{4 \times 3} = \frac{6}{12} = \frac{1}{2}\)
3. מכנה משותף 6: \(\frac{2}{6} + \frac{1}{6} = \frac{3}{6} = \frac{1}{2}\)
4. מכנה משותף 12: \(\frac{10}{12} - \frac{3}{12} = \frac{7}{12}\)
5. \(\frac{2}{3} ÷ \frac{1}{6} = \frac{2}{3} \times \frac{6}{1} = 4\)
6. \(0.375 = \frac{375}{1000} = \frac{3}{8}\)
7. \(\frac{7}{3} + \frac{5}{4} = \frac{28}{12} + \frac{15}{12} = \frac{43}{12} = 3\frac{7}{12}\)
8. \(\frac{4}{5} \times \frac{5}{2} = \frac{20}{10} = 2\)

אחוזים ויחסים

1. \(80 \times \frac{25}{100} = 20\)
2. \(\frac{30}{120} \times 100 = 25\%\)
3. 14 בנים מתוך 32: \(\frac{14}{32} \times 100 = 43.75\%\)

2. \(\sin^2 \beta = 1 - \cos^2 \beta = 1 - \frac{16}{25} = \frac{9}{25}\), \(\sin \beta = \frac{3}{5}\)

4. \(\cos 60° = \frac{1}{2}\)
5. \(\tan 45° = 1\)
6. אם \(\tan \gamma = \frac{3}{4}\), אז \(\sin \gamma = \frac{3}{5}\), \(\cos \gamma = \frac{4}{5}\)

פרק 6: הסתברות וסטטיסטיקה

הסתברות בסיסית

1. 3 מספרים זוגיים (2,4,6) מתוך 6: \(P = \frac{3}{6} = \frac{1}{2}\)
2. 4 מלכים מתוך 52 קלפים: \(P = \frac{4}{52} = \frac{1}{13}\)
3. \(P = \frac{6}{10} = \frac{3}{5}\)
4. דרכים: (2,6), (3,5), (4,4), (5,3), (6,2) = 5 דרכים מתוך 36. \(P = \frac{5}{36}\)
5. \(P(A \cap B) = P(A) \times P(B) = 0.3 \times 0.4 = 0.12\)

קומבינטוריקה

1. \(5! = 5 \times 4 \times 3 \times 2 \times 1 = 120\)
2. \(4! = 24\) דרכים
3. \(C(8,3) = \frac{8!}{3! \times 5!} = \frac{8 \times 7 \times 6}{3 \times 2 \times 1} = 56\)
4. \(P(7,3) = \frac{7!}{4!} = 7 \times 6 \times 5 = 210\)
5. \(4! = 24\) מילים

פרק 7: חשבון דיפרנציאלי

נגזרות

1. \(f'(x) = 4x^3\)
2. \(g'(x) = 6x + 5\)
3. \(h(x) = x^{-3}\), \(h'(x) = -3x^{-4} = -\frac{3}{x^4}\)
4. \(f(x) = 2x^{1/2} - 4x^{-1}\), \(f'(x) = x^{-1/2} + 4x^{-2} = \frac{1}{\sqrt{x}} + \frac{4}{x^2}\)
5. \(f'(x) = 3x^2 - 12x + 9\), \(f'(2) = 12 - 24 + 9 = -3\)

אינטגרלים

1. \(\int x^3 dx = \frac{x^4}{4} + C\)
2. \(\int (2x + 5) dx = x^2 + 5x + C\)
3. \(\int_0^3 x^2 dx = \left[\frac{x^3}{3}\right]_0^3 = \frac{27}{3} - 0 = 9\)
4. \(\int (x^3 - 2x + 1) dx = \frac{x^4}{4} - x^2 + x + C\)
5. \(\int_1^4 x^{-1/2} dx = [2\sqrt{x}]_1^4 = 2(2) - 2(1) = 2\)
